# Sistema de Gestión de Biblioteca (Java)

## Descripción
Aplicación de consola desarrollada en Java que simula la gestión de una biblioteca.
El programa permite crear una biblioteca, gestionar un catálogo de libros y exportar los datos a un fichero.

## Funcionalidades
- Crear una biblioteca
- Crear un catálogo de libros
- Añadir libros al catálogo
- Mostrar la información de todos los libros
- Exportar el catalogo a un fichero (libros.obj)

## Tecnologías utilizadas
- Java
- Programación Orientada a objetos (POO)
